LeetCode 41. First Missing Positive

Description:

Given an unsorted integer array, find the first missing positive integer.
Your algorithm should run in O(n) time and uses constant space.

Example:

For example,
Given [1,2,0] return 3,
and [3,4,-1,1] return 2.

分析：

虽然这道题的题意很简单，但其实这道题还是有些难度的。
首先我们知道一个数放在正确位置上表示：该数的值要比其位置大1，例如，数值1要放在位置0上，数值2要放在位置1上。
所以我们要把位置“放错”的两个数交换过来，从这一点出发，nums[i]要放在nums[i] - 1这个位置上，即要把nums[i]和nums[nums[i] - 1]这两个数交换位置，可利用swap函数解决该问题。

最后，再一次遍历找出第一个缺失的数。

例子：

举个例子：数组[3, 4, -1, 1]

  • 第一次循环：
    nums[2] != nums[0]，
    因此交换：nums[2] = 3；nums[0] = -1
    数组变为[-1, 4, 3, 1].
  • 第二次循环：
    nums[0] = -1 < 0，因此不交换
  • 第三次循环：
    nums[3] != nums[1]
    因此交换：nums[3] = 4；nums[1] = 1
    数组变为[-1, 1, 3, 4].
  • 第四次循环：
    nums[0] != nums[1]
    因此交换：nums[0] = 1；nums[1] = -1
    数组变为[1, -1, 3, 4].
  • 第五次循环：
    nums[2] == 3，if语句不执行
  • 第六次循环：
    nums[3] == 4，if语句不执行

最后循环得出2不在属于它的位置1上，故返回2。
